== Description ==

= Add quick-views to your WooCommerce store. =

The Quickshop plugin adds a new 'Quickshop' button to each product listing, enabling users to view detailed product information and purchase items directly from a modal lightbox without leaving the page.
      	
Both the product-views and Quickshop buttons are fully customizable, and use the standard WooCommerce template system.

This plugin is **WooCommerce 2.x compatible**.

== Installation ==

1. Upload the `woocommerce-quickshop` folder and all its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. After activating the plugin, you can configure it by going to WooCommerce > Settings > Quickshop.
